# Changelog

All notable changes to the WhyStrohm Content Infrastructure Audit skill.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/). This project uses [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-26

### Added
- Two-phase content audit flow (diagnostic + live rewrite)
- 5-layer scoring framework: Vocabulary, Structure, Proof Density, Voice Consistency, Buyer Alignment
- Website scraping for voice analysis (homepage, about page, blog)
- Score-first report format with visual bar display
- Before/after comparison with per-layer deltas
- Full framework reveal (methodology given away by design)
- Configurable CTA template with booking link
